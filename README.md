# Roopesh Rao S — Portfolio

> Live at 👉 **[roopeshraos7.github.io](https://roopeshraos7.github.io)**

A fully animated personal portfolio built with pure HTML, CSS, and JavaScript — no frameworks, no build tools. Just drop it on GitHub Pages and it works.

---

## 📁 Project Structure

```
roopeshraos7.github.io/
├── index.html        # Main HTML — all sections and content
├── css/
│   └── style.css     # All styles, CSS variables, animations, responsive
└── js/
    └── main.js       # Cursor, canvas, particles, typewriter, counters, scroll reveal
```

---

## ✨ Features

- 🎯 Custom magnetic cursor with lagging ring
- 🌈 Glitch + gradient shift effect on hero title
- ⌨️ Typewriter cycling through 6 role phrases
- 🌊 Animated canvas background — moving particles + subtle grid
- 🎈 Floating hero particles with randomised paths
- 🔢 Animated stat counters on page load
- 📜 Scroll reveal — sections animate in from left, right, and bottom
- 💜 Hover effects on all cards — lift, border glow, shimmer overlays
- 📱 Fully responsive for mobile

## 🚀 Deployment

This site is hosted on **GitHub Pages** — no build step needed.

To deploy your own copy:

```bash
# 1. Clone or fork this repo
git clone https://github.com/roopeshraos7/roopeshraos7.github.io.git

# 2. Make your edits

# 3. Push to main branch
git add .
git commit -m "update portfolio"
git push origin main
```

GitHub Pages automatically serves `index.html` from the root of the repo.

---

## ✏️ Customisation Guide

| What to change | Where to edit |
|---|---|
| Colors / fonts / spacing | `css/style.css` → `:root` variables |
| Typewriter phrases | `js/main.js` → `phrases` array |
| Stat counter values | `js/main.js` → `animCount(...)` calls |
| Content / sections | `index.html` |
| Number of canvas particles | `js/main.js` → `dots` array length |
